Hotel Data Flow Demo

Pipeline completo de datos para hoteles: PMS → Python → SQL → Dashboard

Inicio Proyecto Contacto

¿Qué hace?

Este proyecto simula el flujo real de datos en un entorno hotelero, desde la entrada manual o automatizada en el PMS hasta la visualización final en dashboards interactivos.

Stack tecnológico

  • Node.js — lógica PMS y API
  • Python — limpieza, transformación y análisis
  • SQL — almacenamiento estructurado
  • Streamlit — visualización interactiva

Características clave

  • Automatización de procesos
  • Importación de datos históricos
  • Diseño modular y escalable
  • Preparado para integración futura con PMS reales
